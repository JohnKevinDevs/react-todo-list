# 📝 React To-Do List

A simple and interactive To-Do List application built with React.  
This project was developed to practice fundamental React concepts such as **components, props, and state**.

---

## 🚀 Features

- Add new tasks
- Display a dynamic task list
- Mark tasks as completed or pending
- Remove tasks
- Clean and responsive interface

---

## 🛠️ Technologies Used

- React
- JavaScript (ES6+)
- JSX
- CSS Modules

---

## 📚 Concepts Applied

- Component-based architecture
- Props for data communication
- State management with hooks
- Event handling in React
